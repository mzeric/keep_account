#  一款web端记账软件

### Features:

* support i18n
* only cost_sharing + sep, total not implemented now
* 群组消费和个人消费分开记录


### Requirements:

* Python2
* Django 1.6
* Sqlite3

That' all. It's base package of Python web environment

### Installation

```

git clone git@github.com/mzeric/keep_account
cd keep_account
./manage.py runserver

```
